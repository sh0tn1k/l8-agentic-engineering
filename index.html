Учебник для инженеров проф-уровня в ИИ

L8+ Agentic Engineering

Учебник для сильных инженеров: от автономии до производственного чертежа. Деривация от «что такое автономия» до собранного производственного чертежа и протокола, не дающего ему устареть — 14 уроков в 6 фазах, каждый с реальным сломанным прогоном, выводом «проблема → решение → новая проблема» и практикумом.

«L8+» — внутреннее обозначение курса, не отраслевой стандарт (признанные таксономии — L0–L5, производны от SAE J3016; см. урок 1). Актуально на 2026-06-23. Frontier-раздел (урок 14) пересмотреть до 2026-09-23. Метод: research-досье → инженерная сборка с verification gate; каждое число и инцидент проверены по первоисточникам, непроверенное — в карантин.

Опорный стержень: сквозная система

Каждый урок добавляет примитив с контрактом интерфейса и инвариантом «что ломается без него». К финалу — собранный производственный чертёж, не 14 разрозненных конспектов.

приём задачи → классификатор → планировщик → маршрутизатор → сборщик контекста → исполнитель(+реестр инструментов) → верификатор(+критик) → управление памятью ; сквозь всё: политики, трассировка, эскалация
  1. Автономия = f(права, надзор), не способности (урок 1)
  2. Lifecycle: машина состояний + idempotency + event log + replay (уроки 3, 9)
  3. Контекст = бюджет внимания + provenance/trust (урок 4)
  4. Agent proposes, system decides; enforcement вне модели (урок 5)
  5. Trust/provenance: недоверенное не исполняется и не пишется как факт (уроки 4, 5, 7, 8)
  6. Rule of Two: ≤2 из {untrusted input, sensitive access, egress} (урок 10)
  7. Verifiability — узкое место масштабирования (уроки 6, 13)
  8. Accountability не передаётся агенту: RACI, accountable=human (урок 12)

Как читать

  • По фазам, линейно — каждая фаза добавляет слой системы; уроки внутри фазы строятся друг на друге.
  • Структура урока (одинаковая у всех 14): разбор инцидента → деривация → карта решений → режимы отказа → вклад в опорный стержень → практикум → вопросы на суждение → глоссарий → первоисточники.
  • Спорное оставлено спорным — где академия/прод/фреймворки расходятся, урок даёт критерий выбора, а не «правильный ответ».
  • Уроки 3, 4, 6, 7, 10 содержат интерактивные виджеты — потрогать механику руками.
  • Шкала зрелости: устоявшееся формирующееся спорное нерешённое
